# Fleischerei Grämmel — Webseite

Website für Meister Grämmel · Fleischereifachgeschäft in Detmold.

## Seiten

- `index.html` — Startseite (Tradition, Handwerk, Wochenmärkte, Wurstbox 24h, Bewertungen, Kontakt)
- `karriere.html` — Stellenanzeige „Ausbildung zum Fleischer (m/w/d)"

## Tech

Statisches HTML, self-contained. Google Fonts via CDN. Logo als Data-URI eingebettet. Keine Build-Tools nötig.

## Lokal ansehen

Einfach `index.html` im Browser öffnen, oder:

```
python3 -m http.server 8000
```

## Deployment

Auf GitHub Pages: Repo-Settings → Pages → Source: `main` Branch, Root-Ordner. URL: `https://<user>.github.io/<repo>/`

## Noch zu liefern

- Echte Fotos (Theke, Wurstauslage, Räucherei, Meister, Laden, Partyservice) — 6–12 Stück
- Impressum + Datenschutzerklärung
- Verifizierte Daten (E-Mail, tarifliche Vergütung, Meisterbrief-Jahr)
- Tatsächliche Wochenmarkt-Termine und Preise
- Echte Bewertungen oder alternative Zitate
